## 使用介绍

 安装脚本管理器

* **Firefox**: 安装 [Greasemonkey](https://addons.mozilla.org/en-UgS/firefox/addon/greasemonkey/)
* **Chrome**: 安装 [Tampermonkey](https://tampermonkey.net/)
